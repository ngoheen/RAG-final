1. Login/Create Profile

Login to your RAGBRAI.com user account

Before you begin registering for the RAGBRAI ride, you must first log in to your RAGBRAI.com user account. The username and password you used to log into the old registration system is no longer valid.

A RAGBRAI.com user account has several benefits.

  • Register for the RAGBRAI ride.
  • You may participate in the RAGBRAI.com forums.
  • Send private messages to other RAGBRAI.com users.
  • Friend other RAGBRAI.com users so that you can follow their activity streams.
  • Track you RAGBRAI.com shop purchases.

If you don’t already have a RAGBRAI.com user account, Create a RAGBRAI.com user account here.
